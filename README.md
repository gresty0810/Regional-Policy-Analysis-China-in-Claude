基本介绍：
  
  基于claude.ai制作的一套中国大陆特定区域特定主题政策检索skill。运行案例见case文件。


主要功能：
  
  针对特定区域 + 特定主题的组合，系统性地自动执行政策检索和校审。


使用方法：

将本skill导入claude及其他可以读取skill的AI窗口中，输入你想要检索的地名+政策主题（例如北京市 人工智能产业），选择时间范围（3/5/10年）和研究目的（多个，默认为综合研究），AI将自动执行检索流程。


skill内含：

skill.md：skill主体文件。

\references\报告撰写规范.md：强制AI依据以上撰写规范撰写报告，包含四项基本原则（只写有来源的内容、行内引注格式、区分事实与分析、政策关联引用格式）、报告结构模板、降AI味自查清单、质量自检清单。

\references\强制检索清单.md：强制AI检索文件中的政策内容，包括国土空间规划文本、近N年的规划文本等（详见文档）。

  
最终成果：
  
  如果使用claude code运行，该skill将在根目录下自动生成以下子目录及5个文件：
  
  \02_政策资料库\01_完整政策库\政策清单索引.xlsx
  
  \02_政策资料库\01_完整政策库\政策全文汇编.md
  
  \02_政策资料库\02_相关性政策库\相关性政策清单索引.xlsx
  
  \02_政策资料库\02_相关性政策库\相关性政策全文汇编.md
  
  \06_研究报告\01_政策趋势报告\宏观环境扫描与政策趋势报告.md
  
  （编号不连续是因为这个skill是我自用的一整套skill的一个组成部分~可以自行修改，请见谅。）


Skill内置的主要流程：
 
  1-用户需求确认。确认用户输入的具体内容，自动生成任务书；确认用户是否需要上传文件。
 
  2-分级网络检索。根据用户输入的地名，自动检索本层级及上层级政策（例如输入北京市，则自动检索北京市和国家级政策。）
  
  3-政策相关度裁剪。AI自动识别检索得到的政策与用户需求的相关性，并进行标注。相关度裁剪口径可自行调整。
  
  4-政策来源校审。对所有检索到的文件名称、发布机构、发布时间和来源链接进行逐级校审和自动修正，通过校审的政策纳入政策库。
  
  5-自动生成政策文本库、政策文本索引、政策文本汇编和政策趋势报告。


成果文件主要内容：
  
  政策清单索引.xlsx：包含政策的序号、政策层级、政策名称、文件编号、发布机构、发布时间、有效期限、100字概要、相关性分类、引用/依据的政策、被引用/后续政策、来源链接、备用来源、全文位置索引
  
  政策全文汇编.md：经来源校审通过的所有检索政策，含核心政策与关联政策。若未能检索到全文，则录入核心条款。
  
  相关性政策清单索引.xlsx：同政策清单索引，为相关性更高的政策。
  
  相关性政策全文汇编.md：同政策全文汇编，为相关性更高的政策。
  
  宏观环境扫描与政策趋势报告.md：AI根据政策内容生成分析报告，遵循报告撰写规范.md文件里的规定。
